# Summary

This is a tube and reflector designed to fit the wormbot robotics system http://wormbot.org

it mounts to the bottom transilluminator arm and provides a more robust reflector than printed paper reflectors.  If white plastic is not used for the print a 3 inch diameter white paper disk should be affixed to the reflector.  The reflector is designed to be attached to the tube with M4 screws and nuts.
